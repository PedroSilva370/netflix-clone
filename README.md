# Netflix Clone

This project is a front-end clone inspired by the Netflix interface, developed during the Alura Frontend Immersion. It reproduces key aspects of a streaming platform experience, including profile selection and dynamic content browsing.

---

## Features

- User profile selection  
- Theme toggle (light/dark mode)  
- Interactive catalog page  
- Content sliders similar to streaming platforms  
- Hover effects for content emphasis  
- Visual notification elements  
- Profile persistence using localStorage  

---

## Technologies

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Font Awesome  
- Google Fonts  

---
